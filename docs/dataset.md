# ภาพรวมชุดข้อมูล (Dataset Overview) IEEE-CIS Fraud Detection | Kaggle

ชุดข้อมูล IEEE-CIS Fraud Detection จะถูกแบ่งออกเป็น 2 ตารางหลัก ซึ่งสามารถนำมารวมกัน (Join) ได้ด้วยคอลัมน์ `TransactionID`

## 1. ตาราง Transaction (ข้อมูลการทำธุรกรรม)

ข้อมูลพื้นฐานที่เกิดขึ้นตอนรูดบัตรหรือสั่งซื้อ ประกอบด้วยกลุ่มคอลัมน์ดังนี้:

- `isFraud`: ตัวแปรเป้าหมาย (Target variable) โดย 1 หมายถึงธุรกรรมทุจริต และ 0 หมายถึงธุรกรรมปกติ
- `TransactionID`: รหัสอ้างอิงเฉพาะของแต่ละธุรกรรม (ใช้สำหรับ Join ตาราง)
- `TransactionDT`: ระยะเวลาที่ผ่านไปจากจุดอ้างอิงที่ระบบกำหนดไว้ (เป็นหน่วย Time delta ไม่ได้ระบุเป็นวันที่/เวลาจริง)
- `TransactionAmt`: จำนวนเงินของธุรกรรมนั้น (สกุลเงิน USD)
- `ProductCD`: รหัสหรือประเภทของสินค้า/บริการ
- `card1 - card6`: ข้อมูลบัตรชำระเงิน เช่น ประเภทบัตร (เครดิต/เดบิต), เครือข่ายบัตร (Visa/Mastercard), รหัสธนาคาร และรหัสประเทศ
- `addr1, addr2`: ข้อมูลที่อยู่ (addr1 มักหมายถึงโซนพื้นที่รหัสไปรษณีย์ และ addr2 คือรหัสประเทศ)
- `dist1, dist2`: ระยะทางที่เกี่ยวข้อง เช่น ระยะห่างระหว่างตำแหน่งที่ส่งบิลกับที่อยู่ IP
- `P_emaildomain, R_emaildomain`: โดเมนอีเมลของผู้ซื้อ (Purchaser) และผู้รับ (Recipient) เช่น gmail.com
- `C1 - C14 (Counting)`: ตัวแปรกลุ่มนับจำนวน เช่น จำนวนที่อยู่ที่ผูกกับบัตรใบนี้ (ความหมายที่แท้จริงถูกปิดบังไว้)
- `D1 - D15 (Time delta)`: ตัวแปรกลุ่มระยะห่างของเวลา เช่น จำนวนวันนับจากการทำธุรกรรมครั้งก่อนหน้า
- `M1 - M9 (Match)`: ข้อมูลการจับคู่ความถูกต้อง (ค่า True/False หรือ หมวดหมู่) เช่น ตรวจสอบว่าชื่อบนบัตรตรงกับที่อยู่จัดส่งหรือไม่
- `V1 - V339 (Vesta Features)`: ตัวแปรเชิงลึกที่ทางบริษัท Vesta สร้างขึ้นมาจากการคำนวณสถิติ การจัดอันดับ และความสัมพันธ์ของเครือข่ายผู้ใช้งาน เป็นกลุ่มคอลัมน์ที่ใหญ่ที่สุดและถูกปิดบังความหมายที่แท้จริงไว้ทั้งหมด

## 2. ตาราง Identity (ข้อมูลระบุตัวตนและอุปกรณ์)

เก็บข้อมูลเชิงเทคนิคของผู้ใช้ (หมายเหตุ: ข้อมูลตารางนี้ไม่ได้มีครบทุกแถว ธุรกรรมบางรายการจะไม่มีข้อมูลส่วนนี้)

- `TransactionID`: รหัสอ้างอิงเพื่อใช้เชื่อมกับตาราง Transaction
- `DeviceType`: ประเภทอุปกรณ์ที่ใช้ทำธุรกรรม (เช่น mobile, desktop)
- `DeviceInfo`: ข้อมูลเฉพาะของอุปกรณ์ (เช่น ยี่ห้อ, รุ่นมือถือ, ระบบปฏิบัติการ)
- `id_01 - id_38`: กลุ่มตัวแปรระบุตัวตนที่เก็บโดยระบบรักษาความปลอดภัย เช่น คะแนนความเสี่ยงของ IP address, สถานะการใช้ Proxy, พฤติกรรมการล็อกอินที่ผิดปกติ, ความละเอียดหน้าจอ หรือเวอร์ชันของเบราว์เซอร์ (ข้อมูลส่วนใหญ่ในกลุ่มนี้ถูกปิดบังความหมายไว้)
